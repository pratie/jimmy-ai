Chat Widget Embed - Zero Gutter Test

Testing Transparent Iframe Embed

This page has a purple gradient background to easily spot any white gaps around the chat widget.

Key Configuration:

  • ✅ data-margin="0" - No outer spacing from embed script
  • ✅ Iframe forced transparent via CSS
  • ✅ Chat window positioned with its own margin (24px from edge)
  • ✅ Background transparent in iframe document

Expected Result:

The chat widget should appear in the bottom-right corner with:

  • NO white rectangle around it
  • Purple gradient visible right up to the chat window border
  • Only the chat window itself has white background
  • Chat window maintains its black border and rounded corners

Troubleshooting:

If you still see white gaps:

  1. Open DevTools → Inspect the iframe element
  2. Check Computed Styles → background should be "transparent"
  3. Check the iframe's parent container for padding/background
  4. Verify the chatbot page CSS has transparent backgrounds
